## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.

## Development & Maintenance

### Database Operations

```bash
npm run db:backup      # Create timestamped database backup
npm run db:check       # Verify database structure and contents
```

### Application Testing

```bash
npm run app:smoke      # Run smoke test for quick health check
```

### Template Management

```bash
npm run templates:setup     # Setup content templates (first time)
npm run templates:populate  # Populate user templates
```

### Development Tools

```bash
npm run dev:seed       # Create test users for development
```

See `scripts/README.md` for detailed documentation of all utility scripts.
